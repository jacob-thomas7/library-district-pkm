# Library District PKM
*Library District: Personal Knowledge Management Method*

The Library District method is a filesystem-based personal knowledge management method focused on letting you *own*, *find*,
and *keep* your files with minimal intrusion. 

This repository explains how to apply this methodology, and supplies tools for implementing it efficiently.

This project is placed under the [Unlicense](LICENSE.md), and thus the methodology and tooling can be modified at the user's
discretion.

## Ideology
The Library District Methodology (LDM) is focused on the belief that your files are just that—*your* files.

As such, LDM uses no bulky software, no enforcements, no custom file formats—just you and your file system. If you want to
keep your files completely private, only on local hard drives, go ahead! Encrypted and placed on a cloud server? Sure thing!
Want to modify the helper scripts? They are completely open source!

Similarly, you should also be able to *find* your files. LDM helps you do just that! Again, there is no huge program provided
to search through all your files for that one document you made in 2013—instead, it seeks to help you *place* your files where
you *will* find it. Want to look at all your current projects, across all topics? Look in the `projects` folder. Long-term
planning documents related to finances? They're all right next to each other!

Finally, it should be easy to *keep* your files, specifically through backups and data transfers. But often, personal files can
end up scattered across many different folders if you use default save locations, making it easy to lose some during transfer.
LDM seeks to help you keep all your files in one spot, while keeping other files out, keeping your backup sizes to a minimum with
only the data you really need.

The LDM's core ideal is that you can *own*, *find*, and *keep* your files—because they are *yours!*
